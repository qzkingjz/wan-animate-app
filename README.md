# 图生动作 - AI视频生成应用

基于阿里云通义万相 wan2.2-animate-move 模型的图生动作视频生成Web应用。

## 功能特性

- 🖼️ 上传人物图片和参考视频
- 🎬 AI自动生成动作视频
- 📊 实时任务状态跟踪
- 📥 视频下载功能
- 🎨 现代化深色主题UI
